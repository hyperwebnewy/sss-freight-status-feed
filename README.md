# SSS Freight status page

The public project-status page for the SSS Freight build, live at
<https://hyperwebnewy.github.io/sss-freight-status-feed/>.

This repo holds the page itself (Vite, vanilla TypeScript) and `status.json`, the
generated data it reads. The data is machine-written by the `sss-freight-management`
repo and overwritten on every publish; do not hand-edit it.

No credentials, no commercial values and no raw planning files live here.

See `CLAUDE.md` for how the two repos fit together, the rules that govern the copy,
and the traps worth knowing before changing anything.
